Setting Up

Use the array of tokens you obtained, either from GetTokens or GrabNewTokens, to create a new instance of the User class:

$user = new \PSN\User($tokens);

You can now use $user to make requests related to the user.

Me()

Me requires no arguments and will return an object containing your account information:

$me = $user->Me();
//you can use var_dump to see the contents of this object
var_dump($me);

Block($PSN)

Block requires one argument, which is the PSN you want to block on the logged in account:

$user->Block("foobar");

Unblock($PSN)

Unblock requires one argument, which is the PSN you want to unblock on the logged in account:

$user->Unblock("foobar");

GetUserActivity($PSN)

GetUserActivity requires one argument, which is the PSN of the user you want to view the activity for. Note that this function won't work for certain users depending on their privacy settings. This function returns a large object with multiple arrays and values that is best viewed using var_dump:

var_dump($user->GetUserActivity("foobar"));
